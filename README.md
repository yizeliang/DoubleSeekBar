DoubleSeekBar
-------------



![效果图](https://i.loli.net/2018/01/10/5a55b852c4abe.gif)



使用方法
=======

```xml

    <cn.yzl.android.doubleseekbar.DoubleSeekBar
        android:layout_width="match_parent"
        android:layout_height="wrap_content" />

```

注意事项
=======

- 最低高度为80,wrapcontent 也为 80
